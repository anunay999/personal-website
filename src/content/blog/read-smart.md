Welcome to Read Smart. This is the short path from zero to personalized reading.

Read Smart extracts an article, checks it against your private memory of what you have already read, and asks Gemini to rewrite the piece so it focuses on the **new** ideas for you.

Everything runs on your machine or through services you directly control. No hidden servers, no surprise data sales.

You will need two API keys — Gemini (for the rewriting) and Mem0 (for the memory). Both are optional; Reader Mode works without them.

Install the extension by cloning the repo, enabling developer mode in Chrome, and loading the `plugin` folder unpacked. Add your keys in settings, and toggle *Smart Rephrase Mode* on any long article.
